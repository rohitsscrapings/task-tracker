A simple Task Tracker web application built using React.  
The app allows users to manage daily tasks efficiently with a clean and responsive interface.

---

## Features

- Add new tasks (empty tasks are not allowed)
- Mark tasks as completed
- Delete tasks
- Filter tasks (All / Active / Completed)
- View total and completed task statistics
- Dark mode toggle
- Responsive design for different screen sizes

---

## Technologies Used

- React
- JavaScript (ES6)
- Vite
- Basic CSS (no frameworks)

---

## How to Run the Application

1. Install dependencies:
npm install

2. Start the development server:
npm run dev

3. Open the app in the browser:
http://localhost:5173

---

## Example Usage

- Type a task name and click **Add Task** or press **Enter**
- Use the checkbox to mark tasks as completed
- Click **Delete** to remove tasks
- Use filter buttons to view tasks
- Toggle dark mode for better viewing
